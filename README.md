# GitHub User Finder

Автор: Рухтина Татьяна

## Описание
Это приложение позволяет искать пользователей GitHub по имени, просматривать результаты и добавлять избранных пользователей. Избранные сохраняются в JSON и могут быть загружены обратно.

## Как пользоваться API
- Введите имя пользователя в поле поиска.
- Нажмите кнопку "Поиск".
- Дважды кликните по результату поиска — пользователь добавится в избранное.
- Для сохранения избранных используйте кнопку "Сохранить избранное".
- Для загрузки сохраняйте из файла — "Загрузить избранное".

## Примеры использования
- Поиск пользователя "octocat".
- Добавление пользователей в избранное.
- Сохранение и загрузка списка.
